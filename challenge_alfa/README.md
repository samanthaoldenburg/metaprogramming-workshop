Challenge Alfa - Intro to Class Macros
======================================

The goal is to create 3 class macros:

1. `cttr_reader` - `attr_reader` for class-level instance variables
2. `cttr_writer` - `attr_writer` for class-level instance variables
3. `cttr_accessor` - `attr_accessor` for class-level instance variables

You can evaluate your solution via

``` sh
bundle exec rspec
```
